# SG-REITs Analysis: Singapore Real Estate Investment Trusts Performance & Valuation

Comprehensive analysis of Singapore-listed REITs focusing on dividend yield, NAV discount/premium, sector exposure, leverage, and total return performance.

## Key Features
- Historical performance tracking (2015–2025)
- NAV discount/premium analysis
- Sector & geographic diversification breakdown
- Dividend sustainability & payout ratio evaluation
- Comparative valuation vs. peers & benchmarks (STI, S-REIT Index)

## Technologies Used
- Python (pandas, numpy, matplotlib, seaborn, plotly)
- Data sources: SGX, Yahoo Finance, REIT company reports

## Portfolio Impact
- Demonstrates financial data wrangling, time-series analysis, visualization, and investment insight generation.
- Useful for equity research, asset management, REIT specialist roles.
